# ZEN 大学 コミュニティ・ガイドブック

**これは、ZEN 大学の Slack コミュニティ内で行われる多様な活動（サークル、勉強会、プロジェクト等）を発見しやすくするための、学生有志による非公式ガイドブックです。**

このプロジェクトの目的は、コミュニティの「発見アクセスコスト」を引き下げ、学生が自身に最適な居場所を見つけられるようにすることです。

---

## 🧭 プロジェクトの基本方針

このガイドブックは、特定の理念と厳格なルールに基づいて運営されています。

- **私たちの憲章 (Charter.md)**

  - プロジェクトの目的、基本原則（序列化の否定）、統治構造（ガバナンス）、権力移譲計画など、プロジェクトの根幹を定義する**最上位の文書**です。
  - [**憲章 (charter.md) を読む**](./charter.md)

- **運用規約 (Operating-Rules.md)**
  - 憲章に基づき、日々の実務（申請窓口、編集テンプレート、違反時の対応、インシデント処理など）を定義する**具体的なルールブック**です。
  - [**運用規約 (operating-rules.md) を読む**](./operating-rules.md)

---

## 🚀 参加方法（ガイドブックへの関わり方）

### 1. 掲載を希望する方 (チャンネル・サークル運営者)

掲載申請は **GitHub Issues** で受け付けています。

申請の前に、必ず **[運用規約 (operating-rules.md)](./operating-rules.md)** を読み、以下の点をご確認ください。

- **第 1 章：窓口と申請プロセス** （公式窓口の定義）
- **第 2 章：編集の型（テンプレート）** （申請に必要な項目）

### 2. プロジェクトに貢献したい方 (編集者・レビュアー)

このプロジェクトは、創設者から「学生自治」へと移行（権力移譲）するプロセス（Phase B, C）を憲章で定めています。

私たちは、ガイドブックの紹介文を執筆する「編集者」や、掲載判断を担う「レビュアー」として協力してくれる仲間を将来的に募集します。

- **現在のステータス:** **Phase A（創設者による暫定統治・MVP 準備中）**
- **協力に興味がある方へ:** まずは [憲章 (charter.md)](./charter.md) の「第 2 章：統治（ガバナンス）設計」をご一読ください。レビュアー募集は「運用規約」に基づき、適切な時期に告知されます。
